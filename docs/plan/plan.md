# 중앙 통제 컴퓨터 구축 계획

## 개요
이 프로젝트는 여러 데스크탑과 노트북, 휴대폰을 중앙 통제하는 서버를 구축하는 프로젝트이다.
서버는 라즈베리파이5 4GB를 사용할 계획이다.

## 기능

### VPN
 - Tailscale 사용

### 서버측 Signaling Server
 - 언어: GO
 - 프레임워크: Gin
 
### 서버측 WoL 브릿지
 - 언어: GO
 - 조건: 데스크탑 전용
 - Signaling Server와 통합

### 클라이언트측 프로그램/앱
 - 언어: GO
 - 프레임워크: Fyne
 - 직접 설치하는 프로그램/앱

#### 원클릭 NFS/SMB 마운트
 - 버튼만 누르면 SMB/NFS (OS에 따라) 자동 마운트 / 언마운트
 - 서버의 특정 디렉터리를 클라이언트에 마운트 하는 것

#### 파일 업/다운로드
 - 위의 마운트용 디렉터리에 업/다운
 - 사실 마운트 해서 넣으면 되지만 귀찮은 경우를 위한 편의기능

#### 원격 화면 렌더링
 - 기술: WebRTC
 - 조건: 윈도우 및 모바일
 - 장치간의 원격 화면 공유 및 제어
 - 서버는 중계만 클라이언트끼리 원격 제어
 
#### 원격 터미널 접속
 - 조건: 리눅스
 - 다른 장치에서 원격으로 터미널 접속

#### 서버 상태 확인
 - 서버의 터미널에 접근
 - 서버의 CPU, GPU, RAM 등의 리소스 상태 확인

## 보안
 - Tailscale ACL
 - Tailscale Whois API
 - 이상 발생 시 즉시 차단 및 연결 해제
